# Scrapy_autoHome

【Scrapy框架实战】下载汽车之家车型图片</br>
</br>
功能点如下↓：</br>
1.使用Scrapy框架</br>
2.使用Scrapy内置的下载文件的方法（图片和文件）</br>
3.Files Pipeline</br>
4.Images Pipeline</br>
</br>
</br>
后期待扩展的功能：</br>
1.使用随机请求头中间件</br>
2.使用ip代理池中间件来解决反爬虫问题</br>
3.考虑异常中断的情况，并设置中断重连模块</br>
4. ...
